# （五）连接查询

含义：连接查询又称为多表查询，当查询的字段来自多个表时，就会用到连接查询。

## 1. 笛卡尔乘积

笛卡尔乘积现象：表1有 `m` 行，表2有 `n` 行，则查询结果有 `m*n` 行。

```sql
SELECT `name`, boyName FROM boys, beauty;
```

发生原因：没有有效的连接条件  
解决办法：添加有效的连接条件  

```sql
SELECT `name`, boyName FROM boys, beauty
WHERE beauty.boyfriend_id=boys.id;
```
